# Student Marks Performance Analysis (Excel)

## Project Overview
This project analyzes student performance data to identify trends across subjects, classes, and gender using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Key Analysis
- Calculated average marks by subject to identify weak areas
- Compared performance between Class 10A and 10B
- Analyzed gender-based performance differences
- Identified failure rates to highlight academic risk

## Key Insights
- Math had the lowest average score among all subjects
- Class 10B performed better overall than Class 10A
- Female students scored significantly higher on average than male students
- A notable percentage of subject attempts resulted in failure

## Learning Outcome
This project helped build a strong foundation in data cleaning, analysis, and insight generation using Excel.
